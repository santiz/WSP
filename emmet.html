Enter key ========================== In HTML and XML documents, Emmet overrides Enter key to insert formatted line breaks between opening and closing tags. In some cases it will break character input (for example, in Japanese language). To disable Enter key handler, simply add the following option in your user's Preferences file: "disable_formatted_linebreak": true Actions shortcuts ========================== Emmet has a number of actions with keyboard shortcuts that may override ones you're using commonly (for example, Ctrl-E or Ctrl-Down).
